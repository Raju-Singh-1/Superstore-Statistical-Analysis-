# Superstore-Statistical-Analysis-
statical analyzing like description, correlation and probability analysis

📖 Overview

This project focuses on performing statistical analysis on a retail dataset to understand patterns in sales, profit, discount, and customer behavior.

The objective is to apply descriptive statistics, correlation analysis, probability, and hypothesis testing to extract meaningful insights from the data.

🎯 Objectives

Analyze central tendency (mean, median, mode)

Measure variability (variance, standard deviation)

Identify relationships between variables (correlation)

Evaluate probability of business events

Understand data distribution (skewness, kurtosis)

📂 Dataset Description

The dataset contains 51,000+ records with 22 columns, including:

Order Details: order_id, order_date, ship_date

Customer Info: customer_name, segment, region

Product Info: category, sub_category, product_name

Metrics: sales, profit, quantity, discount, shipping_cost

🛠 Tools Used

Excel → Statistical calculations & analysis

SQL (MySQL) → Data querying

Power BI → Visualization (optional)

📊 Statistical Techniques Used
1. Descriptive Statistics

Mean, Median, Mode

Standard Deviation, Variance

Range, Coefficient of Variation

Skewness & Kurtosis

2. Correlation Analysis

Sales vs Profit

Discount vs Profit

Quantity vs Sales

3. Probability Analysis

Probability of loss-making orders

Probability of high-value sales

Probability of discount usage

🎯 Conclusion

This project demonstrates how statistical techniques can be used to:

Understand business performance

Identify risk factors (loss-making orders)

Support data-driven decision making
